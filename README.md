# Cadastro Simples de Alunos em C

Este exercício tem como objetivo praticar **entrada e saída de dados em linguagem C**, utilizando o paradigma procedural.

O programa realiza um cadastro simples de um aluno, solicitando ao usuário as seguintes informações:

- Nome
- Idade
- Altura
- Matrícula

Após a entrada dos dados, o programa exibe um **resumo do cadastro** no terminal utilizando `printf` e diferentes **especificadores de formato** (`%s`, `%d`, `%f`).

## Conceitos praticados

- Entrada de dados com `scanf` e `fgets`
- Saída de dados com `printf`
- Uso de variáveis de diferentes tipos (`char`, `int`, `float`)
- Utilização de especificadores de formato
- Estrutura básica de um programa em C

## Compilação 

gcc cadastro-alunos.c -o cadastro
